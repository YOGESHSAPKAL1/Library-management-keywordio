# Library-management-keywordio

# Requirements

pip install django<br>
pip install pillow

# Features

● Add a Book<br>
● Update an entry of a book<br>
● Delete a book<br>
● Get all books<br>
● Admin login<br>
● Admin signup<br>
● Add students<br>
● Student list<br>
● Student view<br>
